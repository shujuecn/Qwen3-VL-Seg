## 0. 先建立传统 CV 到 VL 的坐标系

原文句意（英文复述）：Traditional CV predicts fixed labels, boxes, or masks from images, while VL tasks condition prediction on natural language.

中文意译：传统 CV 通常从图像预测固定类别、框或 mask；VL 任务则把自然语言也作为条件输入。

我的注释：这是最重要的入口差异。你以前的类别 head 是固定维度，现在“类别”可能是一整句文本。

原文句意（英文复述）：Visual grounding means linking a language expression to an image region.

中文意译：视觉 grounding 指的是把一句语言描述对应到图像中的某个区域。

我的注释：如果输出是 bbox，接近开放词表检测；如果输出是 mask，就是 referring segmentation。

原文句意（英文复述）：Referring segmentation asks the model to output pixel-level masks for targets described by language.

中文意译：指代分割要求模型根据语言描述输出对应目标的像素级 mask。

我的注释：这可以理解为“语言条件实例分割”，但目标类别和描述方式不固定。

## 1. Abstract

原文句意（英文复述）：Open-world referring segmentation maps unconstrained language expressions to precise pixel-level regions.

中文意译：开放世界指代分割要把不受固定类别限制的自然语言描述映射到精确像素区域。

我的注释：这里的 open-world 不只是类别开放，也包括表达方式、属性、空间关系和场景开放。

原文句意（英文复述）：Modern MLLMs are good at open-world visual grounding but usually stop at bounding boxes.

中文意译：现代多模态大模型已经擅长开放世界视觉定位，但通常只输出 bbox。

我的注释：Qwen3-VL 能“听懂你说的是谁”，但 bbox 不是精细分割结果。

原文句意（英文复述）：Sparse boxes are not enough for dense visual prediction.

中文意译：稀疏的框坐标不足以完成像素级密集预测。

我的注释：bbox 丢掉了边界形状、孔洞、细长结构和遮挡轮廓。

原文句意（英文复述）：Some methods predict contour coordinates, while others rely on external segmenters such as SAM.

中文意译：已有方法要么预测轮廓坐标，要么依赖 SAM 这类外部分割模型。

我的注释：轮廓 token 容易离散化；SAM 路线效果强但系统更重。

原文句意（英文复述）：Qwen3-VL-Seg treats the MLLM-predicted box as a semantically grounded structural prior.

中文意译：Qwen3-VL-Seg 把 MLLM 预测的 bbox 当作带语义的结构先验。

我的注释：这是全文核心。box 不是最终答案，而是 mask decoder 的条件。

原文句意（英文复述）：The decoder converts box-level grounding into pixel-level segmentation.

中文意译：decoder 把框级定位结果转换为像素级分割结果。

我的注释：传统 CV 类比是“先检测，再在 ROI 内分割”，但检测来自语言模型。

原文句意（英文复述）：The lightweight decoder combines multi-scale spatial injection, spatial-semantic query construction, high-resolution pixel fusion, and mask-aware refinement.

中文意译：轻量 decoder 结合了多尺度空间特征注入、语义-空间 query 构造、高分辨率像素融合和 mask 感知 refinement。

我的注释：可以粗略类比 FPN + DETR query + U-Net skip + cascade refinement。

原文句意（英文复述）：The added decoder has about 17M parameters, a small fraction of the base model.

中文意译：新增 decoder 约 1700 万参数，只占基础模型很小比例。

我的注释：它不是再挂一个 SAM，而是在 Qwen3-VL 上加小分割头。

原文句意（英文复述）：The paper constructs SA1B-ORS with category-oriented and descriptive subsets.

中文意译：论文构造 SA1B-ORS 数据集，包含类别导向和描述导向两个子集。

我的注释：CoRS 训练“分割某类别”，DeRS 训练“分割符合这段描述的实例”。

原文句意（英文复述）：The paper also curates ORS-Bench for in-distribution and out-of-distribution evaluation.

中文意译：论文还构造 ORS-Bench，用于分布内和分布外评估。

我的注释：OOD 包括新类别、极端尺度、遮挡、弱光、复杂描述和风险场景。

## 2. Introduction

原文句意（英文复述）：MLLM progress has expanded visual understanding beyond image-level QA and dialogue.

中文意译：MLLM 的发展让视觉理解不再局限于整图问答和多模态对话。

我的注释：模型开始能定位局部区域，甚至输出结构化坐标。

原文句意（英文复述）：Many vision-language tasks require associating linguistic concepts with specific image evidence.

中文意译：很多 VL 任务需要把语言概念与图像中的具体证据区域关联起来。

我的注释：比如“右边穿红衣的人”必须落到局部像素，而不是整图标签。

原文句意（英文复述）：This association is called visual grounding.

中文意译：这种语言到图像区域的关联能力叫 visual grounding。

我的注释：grounding 是后续 referring segmentation 的基础。

原文句意（英文复述）：Frontier MLLMs can localize arbitrary visual entities using natural-language instructions.

中文意译：前沿 MLLM 可以根据自然语言指令定位任意视觉实体。

我的注释：这相当于检测类别由语言指定，而不是由固定分类头指定。

原文句意（英文复述）：However, these outputs are usually boxes, which are coarse spatial localization.

中文意译：但是这些输出通常是 bbox，只提供粗粒度空间定位。

我的注释：box 能表达“在哪里”，但不能表达“边界是什么形状”。

原文句意（英文复述）：Robotics, medical analysis, and fine-grained editing require boundary-aware spatial reasoning.

中文意译：机器人、医学分析和精细图像编辑都需要边界感知的空间推理。

我的注释：你的舌体分割任务就属于这种需要精确 mask 的场景。

原文句意（英文复述）：A natural next step is moving from box localization to mask prediction.

中文意译：自然的下一步就是从 bbox 定位走向 mask 预测。

我的注释：这就是 REC 到 RES 的升级。

原文句意（英文复述）：One route couples MLLMs with external segmentation models like SAM.

中文意译：一种路线是把 MLLM 与 SAM 这类外部分割模型耦合起来。

我的注释：MLLM 出 prompt/box，SAM 出 mask，是常见两阶段系统。

原文句意（英文复述）：This route is effective but introduces architectural and deployment overhead.

中文意译：这种路线有效，但增加了结构和部署成本。

我的注释：显存、延迟、依赖、模型管理都会增加。

原文句意（英文复述）：Another route equips MLLMs with lightweight dense heads or autoregressive mask representations.

中文意译：另一种路线是给 MLLM 加轻量 dense head，或让模型自回归生成 mask 表示。

我的注释：这是 SAM-free 路线，追求系统一体化。

原文句意（英文复述）：Text-based mask representations suffer from discretization or representation bottlenecks.

中文意译：用文本形式表达 mask 容易遇到离散化和表示瓶颈。

我的注释：用 token/坐标串描述连续边界，本质上不如 feature map 上的 dense decoder 自然。

原文句意（英文复述）：Lightweight dense decoders struggle when they only use low-resolution latent features.

中文意译：如果轻量 dense decoder 只用低分辨率 latent feature，也很难恢复精细边界。

我的注释：这就是论文要加高分辨率图像分支和多尺度特征的原因。

原文句意（英文复述）：Existing benchmarks are often closed-set or limited in vocabulary.

中文意译：现有 benchmark 往往是闭集或词表有限。

我的注释：RefCOCO 很重要，但不能完全代表真实开放世界。

原文句意（英文复述）：The key question is whether MLLM grounding can be transformed into pixel masks without heavy external segmenters.

中文意译：核心问题是能否不用重型外部分割器，把 MLLM 的 grounding 能力转换为像素 mask。

我的注释：这句话就是论文的问题定义。

原文句意（英文复述）：The paper answers this by using the predicted box as a structural prior throughout the decoder.

中文意译：论文的回答是：在 decoder 全流程中把预测 bbox 当作结构先验。

我的注释：box 会参与 query、pixel fusion 和 refinement，而不是只作为辅助输出。

原文句意（英文复述）：The paper also builds large-scale open-world training and evaluation data.

中文意译：论文还构建了大规模开放世界训练和评估数据。

我的注释：方法和数据是这篇文章的两条主线。

## 4. Method Overview

原文句意（英文复述）：The decoder is built on top of a pretrained MLLM.

中文意译：decoder 架在预训练 MLLM 之上。

我的注释：不是从零训练一个分割模型，而是复用 Qwen3-VL 的视觉-语言能力。

原文句意（英文复述）：The model uses the grounded box as a prior for dense mask decoding.

中文意译：模型使用 grounded bbox 作为 dense mask 解码的先验。

我的注释：box 把整图搜索问题变成局部精修问题。

原文句意（英文复述）：The decoder enriches intermediate ViT features with lightweight spatial adapters.

中文意译：decoder 用轻量空间 adapter 增强 ViT 中间层特征。

我的注释：传统 CV 类比 FPN lateral + 局部卷积增强。

原文句意（英文复述）：The decoder combines these features with multimodal visual embeddings.

中文意译：decoder 将这些视觉特征与多模态视觉 embedding 结合。

我的注释：这样 feature map 既有空间细节，也带语言对齐后的语义。

原文句意（英文复述）：The decoder constructs object queries from language semantics and box geometry.

中文意译：decoder 用语言语义和 box 几何共同构造 object query。

我的注释：query 不再只是可学习向量，而是“这个语言目标 + 这个空间位置”。

原文句意（英文复述）：The decoder uses a soft box gate to fuse shallow image features.

中文意译：decoder 用 soft box gate 融合浅层图像特征。

我的注释：浅层图像提供边缘纹理，box gate 抑制背景噪声。

原文句意（英文复述）：The first-pass mask is fed back to refine the query for a second prediction.

中文意译：第一轮 mask 会反馈回 query，用于第二轮预测。

我的注释：这是 coarse-to-fine refinement。

## 5. Multi-scale Spatial Feature Injection

原文句意（英文复述）：High-level MLLM features are too coarse for accurate boundary delineation.

中文意译：高层 MLLM 特征过粗，不足以精确描绘边界。

我的注释：这和传统分割不能只用最后一层 feature 是同一个问题。

原文句意（英文复述）：Intermediate visual features are projected into the decoder hidden dimension.

中文意译：中间视觉特征会被投影到 decoder 的隐藏维度。

我的注释：`1x1 Conv` 做通道对齐，类似 FPN。

原文句意（英文复述）：A depthwise convolution branch injects local spatial bias.

中文意译：depthwise convolution 分支注入局部空间偏置。

我的注释：ViT 擅长全局关系，卷积补局部边缘和纹理。

原文句意（英文复述）：A learnable scalar initialized near zero stabilizes the adapter.

中文意译：一个近零初始化的可学习缩放因子用于稳定 adapter。

我的注释：新模块一开始近似不生效，避免破坏预训练特征。

原文句意（英文复述）：The adapted multi-scale features are concatenated and fused.

中文意译：适配后的多尺度特征会拼接并融合。

我的注释：这是为了同时保留低层空间细节和高层语义。

原文句意（英文复述）：Projected multimodal visual embeddings are reshaped into a 2D feature map.

中文意译：投影后的多模态视觉 embedding 被 reshape 成二维特征图。

我的注释：LLM 内部是 token 序列，分割 decoder 需要 H x W feature map。

原文句意（英文复述）：The memory feature combines multimodal semantics, spatial features, and positional encoding.

中文意译：memory 特征融合了多模态语义、空间特征和位置编码。

我的注释：这是 transformer decoder 后续要 attend 的 dense memory。

## 6. Spatial-Semantic Query Construction

原文句意（英文复述）：The object query should encode target semantics and instance-level spatial identity.

中文意译：object query 应同时编码目标语义和实例级空间身份。

我的注释：同类多实例时，只知道类别远远不够。

原文句意（英文复述）：The grounded box is explicitly used during query construction.

中文意译：grounded bbox 被显式用于 query 构造。

我的注释：这就是 “spatial-semantic query” 中 spatial 的来源。

原文句意（英文复述）：The box position and scale are encoded using Fourier positional encoding.

中文意译：box 的位置和尺度用 Fourier 位置编码表示。

我的注释：连续坐标经过高频编码后，MLP 更容易学习几何条件。

原文句意（英文复述）：The width and height use a log-scale transform.

中文意译：box 宽高经过 log-scale 变换。

我的注释：检测框回归里也常用 log 宽高，因为尺度变化很大。

原文句意（英文复述）：The initial query adds box embedding and projected segmentation-token features.

中文意译：初始 query 由 box embedding 和投影后的 segmentation token 特征相加得到。

我的注释：这意味着 query 同时知道“目标是什么”和“大概在哪”。

原文句意（英文复述）：Transformer decoder layers attend the query to the global memory.

中文意译：transformer decoder 层让 query 去 attend 全局 memory。

我的注释：类似 DETR/Mask2Former 的 query 从 dense feature 中读取目标信息。

## 7. Box-Guided High-Resolution Pixel Fusion

原文句意（英文复述）：Shallow image features provide boundary details missing from high-level features.

中文意译：浅层图像特征提供高层特征缺失的边界细节。

我的注释：传统分割中的 skip connection 就是这个目的。

原文句意（英文复述）：Directly fusing shallow features may introduce background clutter.

中文意译：直接融合浅层特征可能引入背景干扰。

我的注释：浅层边缘太多，不加限制会把无关纹理带进来。

原文句意（英文复述）：The grounded box is enlarged by 15 percent to tolerate localization errors.

中文意译：grounded bbox 会扩大 15%，以容忍定位误差。

我的注释：预测框可能略小，外扩能避免切掉目标边界。

原文句意（英文复述）：The soft gate is built from sigmoid functions along the four box boundaries.

中文意译：soft gate 由 box 四条边界上的 sigmoid 函数组成。

我的注释：这比 hard crop 平滑，可微，也有边界缓冲。

原文句意（英文复述）：The paper sets the gate sharpness parameter to 20.

中文意译：论文把 gate 的边界陡峭系数设为 20。

我的注释：越大越接近硬框，越小越柔和。

原文句意（英文复述）：When multiple boxes exist, the model takes the maximum over their gates.

中文意译：当存在多个 box 时，模型对多个 gate 取空间最大值。

我的注释：这支持“所有 person”这种多实例指令。

原文句意（英文复述）：Upsampled visual features are fused with gated shallow features.

中文意译：上采样后的视觉特征会与 gated 浅层特征融合。

我的注释：语义来自高层，边界来自浅层，box gate 控制范围。

## 8. Iterative Mask-Aware Query Refinement

原文句意（英文复述）：A single mask prediction is often insufficient for thin structures or cluttered scenes.

中文意译：单轮 mask 预测常常不足以处理细结构或杂乱场景。

我的注释：这也是很多分割模型做 cascade/refine 的原因。

原文句意（英文复述）：The decoded query generates dynamic kernels for an initial mask.

中文意译：解码后的 query 生成 dynamic kernel，用来预测初始 mask。

我的注释：dynamic kernel 可以理解为“目标专属分割参数”。

原文句意（英文复述）：The soft mask pools target-aware features from the pixel feature map.

中文意译：soft mask 会从 pixel feature map 中池化目标感知特征。

我的注释：用第一轮预测区域做 attention pooling。

原文句意（英文复述）：The pooled target feature is projected and added back to the query.

中文意译：池化得到的目标特征经过投影后加回 query。

我的注释：query 看到第一轮 mask 证据后再更新。

原文句意（英文复述）：The refined query predicts a second-pass mask.

中文意译：refined query 会预测第二轮 mask。

我的注释：第二轮用于修边界、补漏和抑制误分。

原文句意（英文复述）：An auxiliary IoU head estimates mask confidence.

中文意译：辅助 IoU head 估计 mask 置信度。

我的注释：这有助于给 mask 排序或过滤低质量预测。

原文句意（英文复述）：DeRS provides descriptive instance-specific samples.

中文意译：DeRS 提供描述导向、实例特定的样本。

中文意译：SA-1B 的 mask 是碎片化、无类别、且没有语言配对的。

中文意译：pipeline 首先识别可靠的可指代实体。

中文意译：开放词表 tag 会经过语义规则过滤。

原文句意（英文复述）：Qwen3-VL-Plus predicts grounded boxes and SAM2 produces coarse masks.

中文意译：Qwen3-VL-Plus 预测 grounding box，SAM2 产生粗 mask。

我的注释：注意这里 SAM2 是数据构造工具，不是最终模型推理组件。

原文句意（英文复述）：MLLM verification checks semantic alignment, coverage, contamination, fragmentation, and omission.

中文意译：MLLM 验证会检查语义对齐、覆盖完整性、非目标污染、碎片化和漏标。

我的注释：这是用 MLLM 做自动质检。

原文句意（英文复述）：DeRS generates discriminative descriptions from category, attributes, state, position, and relations.

中文意译：DeRS 从类别、属性、状态、位置和关系生成有区分度的描述。

我的注释：这是训练模型理解复杂指代表达的关键。

原文句意（英文复述）：Generated descriptions are verified by asking Qwen3-VL-Plus to localize the target again.

中文意译：生成的描述会再次交给 Qwen3-VL-Plus 定位目标来验证。

原文句意（英文复述）：Samples with localization IoU below 0.8 are discarded.

中文意译：重新定位 IoU 低于 0.8 的样本会被丢弃。

我的注释：这是 DeRS 的自动质量过滤阈值。

原文句意（英文复述）：Stage 1 focuses on segmentation-centric adaptation.

中文意译：Stage 1 重点做分割中心的适配。

我的注释：这阶段会让模型更偏 dense perception。

原文句意（英文复述）：Stage 1 trains LoRA adapters, the vision encoder, and the mask decoder.

中文意译：Stage 1 训练 LoRA adapter、vision encoder 和 mask decoder。

我的注释：基础 LLM 权重冻结，避免 4B 主体训练成本过高。

原文句意（英文复述）：The model is supervised to generate structured outputs containing boxes and mask placeholders.

中文意译：模型被监督生成包含 bbox 和 mask placeholder 的结构化输出。

原文句意（英文复述）：Stage 2 merges LoRA and fine-tunes the LLM backbone with the mask decoder.

中文意译：Stage 2 合并 LoRA 后微调 LLM backbone 和 mask decoder。

我的注释：这一步风险更高，所以学习率很小。

原文句意（英文复述）：Stage 2 mixes segmentation, general multimodal understanding, and reasoning data.

中文意译：Stage 2 混合分割、通用多模态理解和推理数据。

我的注释：这是为了防止模型只会分割而丢掉 VQA/OCR/推理能力。

原文句意（英文复述）：The prompt follows Qwen3-VL ChatML format.

中文意译：prompt 遵循 Qwen3-VL 的 ChatML 格式。

我的注释：图像 token 和文字指令都放在对话模板里。

原文句意（英文复述）：The output schema contains bbox_2d, label, and mask fields.

中文意译：输出 schema 包含 `bbox_2d`、`label` 和 `mask` 字段。

我的注释：这和当前仓库 JSON 格式基本一致。

原文句意（英文复述）：The mask field is a placeholder that bridges language output and binary mask generation.

中文意译：`mask` 字段是连接语言输出和二值 mask 生成的 placeholder。

我的注释：它不是把 mask 以文本打印出来，而是给 decoder 一个对应目标的锚点。

中文意译：基础模型是 Qwen3-VL-4B，新增 decoder 约 17M 参数。

中文意译：Stage 1 使用 rank 32 的 LoRA，训练 10000 次迭代。

我的注释：当前舌体仓库不建议直接照搬，因为数据只有 300 张。

原文句意（英文复述）：The segmentation objective combines BCE and Dice loss.

中文意译：分割目标由 BCE 和 Dice loss 组成。

我的注释：这是传统二值分割很常见的组合。

原文句意（英文复述）：The model is evaluated on RefCOCO, RefCOCO+, and RefCOCOg for referring segmentation.

## 13. Ablation and Conclusion

原文句意（英文复述）：Freezing the vision encoder degrades performance, especially on high-threshold metrics.

原文句意（英文复述）：Removing the shallow image branch also hurts mask precision and completeness.

我的注释：高分辨率纹理对边界恢复有贡献。

中文意译：完整模型最好，因为各组件是互补的。

我的注释：box 定位、语言语义、多尺度空间和浅层边界分别解决不同问题。
